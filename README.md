## 1. 問題設定

### ハミルトニアン

2次元正方格子上の強磁性イジング模型のハミルトニアンは以下で与えられる：

$$H = -J \sum_{\langle i,j \rangle} s_i s_j$$

| 記号 | 意味 |
|------|------|
| $s_i = \pm 1$ | 各格子点 $i$ のスピン変数 |
| $J > 0$ | 強磁性結合定数 |
| $\langle i,j \rangle$ | 最近接サイトのペア |


### 系のサイズと境界条件

系は $L \times L$ の正方格子とし、**周期境界条件**を課す：

$$s_{L+1,\, j} = s_{1,\, j}, \qquad s_{i,\, L+1} = s_{i,\, 1}$$

これにより格子はトーラス（ドーナツ面）と同相になり、表面効果を取り除くことができる。

### 厳密解（Onsager解）

2次元イジング模型は Onsager（1944）により厳密に解かれており、転移温度の厳密値は

$$\frac{k_B T_c}{J} = \frac{2}{\ln(1+\sqrt{2})} \approx 2.2692$$

で与えられる。本課題ではこの厳密値との比較によりモンテカルロ計算の精度を検証する。

## 2. モンテカルロ法

### メトロポリスアルゴリズム

カノニカル集団（温度 $T$ 固定）でのサンプリングには**メトロポリスアルゴリズム**を用いる。1回の更新試行は以下のステップからなる：

1. サイト $(i,j)$ をランダムに選ぶ。
2. スピン反転 $s_{i,j} \to -s_{i,j}$ に伴うエネルギー変化を計算する：

$$\Delta E = 2 J s_{i,j} \sum_{\delta} s_{i+\delta}$$

（$\delta$ は4つの最近接方向）

3. 受理確率でスピン反転を受理または棄却する：

$$P_{\rm accept} = \min\!\left(1,\; e^{-\Delta E / k_B T}\right)$$

### 1モンテカルロステップ（1 MCS）

> **定義：** 1 MCS = 全 $L^2$ サイトの状態更新を1回ずつ試みること

- 1サイトの更新コスト：最近接4サイトの参照のみ → $O(1)$
- **1 MCS 全体のコスト：$O(L^2)$**
- $N_{\rm step}$ ステップの全計算時間：$O(N_{\rm step} \cdot L^2)$

$L$ を2倍にするごとに計算時間が約4倍になることを数値的に確認する。

### 熱化と測定

| フェーズ | 内容 |
|----------|------|
| 熱化（thermalization） | $N_{\rm therm}$ MCS、平衡分布に収束させる |
| 測定（measurement） | $N_{\rm meas}$ MCS、物理量を統計平均する |

## 3. 観測量

### 磁化と磁化率

$$m = \frac{1}{L^2} \sum_{i} s_i$$

$$\chi = \frac{L^2}{k_B T}\Bigl(\langle m^2 \rangle - \langle |m| \rangle^2\Bigr)$$

磁化率 $\chi$ は相転移点付近で発散し、有限系では $L$ に依存したピークを示す。

### ビンダーキュムラント

有限サイズスケーリングに用いる**ビンダーキュムラント（Binder cumulant）**：

$$U_L = 1 - \frac{\langle m^4 \rangle}{3\langle m^2 \rangle^2}$$

| 相 | $U_L$ の漸近値 |
|----|----------------|
| 秩序相（低温） | $U_L \to 2/3$ |
| 無秩序相（高温） | $U_L \to 0$ |

---

## 4. 有限サイズスケーリング

### ビンダークロスによる $T_c$ 決定

スケーリング理論によれば、ビンダーキュムラントは転移点付近で

$$U_L(T) = f\!\left[(T - T_c) L^{1/\nu}\right]$$

というスケーリング形式をとる（$\nu$：相関長の臨界指数）。

この性質から、**異なるシステムサイズ $L$ での $U_L(T)$ 曲線は $T = T_c$ で交差する（ビンダークロス）**。

```
U_L
 2/3 |         __________
     |        /   L=64
     |       /  __________
     |      / /   L=32
     |   __/ /
     |  /   /  ← 交差点 = T_c
     | /   /
   0 |____________________________→ T
              T_c ≈ 2.269
```

この交点を外挿することで $T_c$ を決定する。

---

## 5. 計算手順のまとめ

1. $L = 8, 16, 32, 64$ 等、複数のシステムサイズについて計算を実行する。
2. 各 $L$、各温度 $T$ に対して $N_{\rm therm}$ MCS 熱化し、$N_{\rm meas}$ MCS 測定する。
3. $m$、$\chi$、$U_L$ を温度の関数としてプロットする。
4. ビンダーキュムラントの交差点から $T_c$ を推定し、Onsager解と比較する。
5. 計算時間の $L$ 依存性を測定し、$O(L^2)$ スケーリングを確認する。
6. 小さな $L$ に対するユニットテストを実施する。

---

## 6. 実装上の注意点

- **周期境界条件**：インデックス計算に `%` 演算（mod）を用いる。
- **初期配置**：高温ではランダム配置、低温では全スピン整列配置が収束を速める。
- **統計誤差**：自己相関時間を考慮し、jackknife 法や bootstrap 法で誤差を推定する。
- **乱数**：メルセンヌ・ツイスタ等の品質の高い擬似乱数生成器を用いる。
- **$O(L^2)$ 確認**：$L$ を2倍にするごとに計算時間が約4倍になることを確認する。
