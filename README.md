## 1. 問題設定

### ハミルトニアン

2次元正方格子上の強磁性イジング模型のハミルトニアンは以下で与えられる：

$$H = -J \sum_{\langle i,j \rangle} s_i s_j$$

| 記号 | 意味 |
|------|------|
| $s_i = \pm 1$ | 各格子点 $i$ のスピン変数 |
| $J > 0$ | 強磁性結合定数 |
| $\langle i,j \rangle$ | 最近接サイトのペア |


### 系のサイズと境界条件

系は $L \times L$ の正方格子とし、**周期境界条件**を課す：

$$s_{L+1,\, j} = s_{1,\, j}, \qquad s_{i,\, L+1} = s_{i,\, 1}$$

これにより格子はトーラス（ドーナツ面）と同相になり、表面効果を取り除くことができる。

> **表面効果とは：** 開放端を持つ格子では端のスピンが内部より隣接数が少なく、実効的に弱い結合領域ができる。これにより見かけの $T_c$ がずれたり、磁化が不均一になるなど、有限サイズ効果と混同される問題が生じる。

#### Onsager解との境界条件の比較

Onsager（1944）の厳密解では、トーラスではなく**円筒（cylinder）境界条件**が用いられている。

> 原論文 p.117：*"a long strip crystal of finite width (n atoms), joined straight to itself around a cylinder"*

| 方向 | Onsager（厳密解） | 本課題（モンテカルロ） |
|---|---|---|
| 横方向 | 周期（リング） | 周期 |
| 縦方向 | 熱力学極限（$n\to\infty$） | 周期（トーラス） |

境界条件は異なるが、**熱力学極限 $L\to\infty$ では同じ $T_c$ を与える**ため、厳密値との比較に問題はない。

### 厳密解（Onsager解）

2次元イジング模型は Onsager（1944）により厳密に解かれており、転移温度の厳密値は

$$\frac{k_B T_c}{J} = \frac{2}{\ln(1+\sqrt{2})} \approx 2.2692$$

この式は原論文付録の式(8.1)：$\sinh(2J/k_BT_c)\sinh(2J'/k_BT_c)=1$ において、正方格子（$J=J'$）の場合に $\sinh(2J/k_BT_c)=1$ を解くことで得られる。本課題ではこの厳密値との比較によりモンテカルロ計算の精度を検証する。

## 2. モンテカルロ法

### メトロポリスアルゴリズム

カノニカル集団（温度 $T$ 固定）でのサンプリングには**メトロポリスアルゴリズム**を用いる。1回の更新試行は以下のステップからなる：

1. サイト $(i,j)$ をランダムに選ぶ。
2. スピン反転 $s_{i,j} \to -s_{i,j}$ に伴うエネルギー変化を計算する：

$$\Delta E = 2 J s_{i,j} \sum_{\delta} s_{i+\delta}$$

（$\delta$ は4つの最近接方向）

3. 受理確率でスピン反転を受理または棄却する：

$$P_{\rm accept} = \min\!\left(1,\; e^{-\Delta E / k_B T}\right)$$

- $\Delta E < 0$（エネルギーが下がる）→ 必ず反転
- $\Delta E > 0$（エネルギーが上がる）→ 確率 $e^{-\Delta E/k_BT}$ で反転（高温ほど上り坂も許容）

**この手続き全体の目的**は、温度 $T$ のボルツマン分布 $P(\{s\}) \propto e^{-H/k_BT}$ に従って配置空間をサンプリングすることである。受理確率がこの形をとるのは、**詳細釣り合い条件**（detailed balance）

$$P(\{s\})\, P_{\rm accept}(\{s\}\to\{s'\}) = P(\{s'\})\, P_{\rm accept}(\{s'\}\to\{s\})$$

を満たすように設計されているためであり、十分な時間後に系がボルツマン分布へ収束することが保証される。

### 1モンテカルロステップ（1 MCS）

> **定義：** 1 MCS = 全 $L^2$ サイトの状態更新を1回ずつ試みること

- 1サイトの更新コスト：最近接4サイトの参照のみ → $O(1)$
- **1 MCS 全体のコスト：$O(L^2)$**
- $N_{\rm step}$ ステップの全計算時間：$O(N_{\rm step} \cdot L^2)$

$L$ を2倍にするごとに計算時間が約4倍になることを数値的に確認する。

### 熱化と測定

| フェーズ | 内容 |
|----------|------|
| 熱化（thermalization） | $N_{\rm therm}$ MCS、平衡分布に収束させる |
| 測定（measurement） | $N_{\rm meas}$ MCS、物理量を統計平均する |

## 3. 観測量

### 磁化と磁化率

$$m = \frac{1}{L^2} \sum_{i} s_i$$

$$\chi = \frac{L^2}{k_B T}\Bigl(\langle m^2 \rangle - \langle |m| \rangle^2\Bigr)$$

磁化率 $\chi$ は相転移点付近で発散し、有限系では $L$ に依存したピークを示す。

### ビンダーキュムラント

有限サイズスケーリングに用いる**ビンダーキュムラント（Binder cumulant）**：

$$U_L = 1 - \frac{\langle m^4 \rangle}{3\langle m^2 \rangle^2}$$

| 相 | $U_L$ の漸近値 |
|----|----------------|
| 秩序相（低温） | $U_L \to 2/3$ |
| 無秩序相（高温） | $U_L \to 0$ |

## 4. 有限サイズスケーリング

### なぜ必要か

無限大の系では相転移はちょうど $T_c$ で起きるが、有限系 $L \times L$ では転移が**温度方向にぼける**。$L$ が大きいほどぼけは小さくなるため、複数の $L$ で計算し $L\to\infty$ へ外挿することで $T_c$ を決める。

### スケーリング仮説

転移点付近では相関長 $\xi \sim |T-T_c|^{-\nu}$ が発散する。有限系では $L$ と $\xi$ の比が振る舞いを決めるため、一般の物理量は

$$\mathcal{O}(T, L) = L^{p/\nu}\, \tilde{f}\!\left[(T-T_c)\,L^{1/\nu}\right]$$

という形をとる。磁化率 $\chi$ などは $L$ 依存の係数 $L^{p/\nu}$ がつくため、異なる $L$ の曲線を直接比較しにくい。

> **Fisher--Barber (1972) との対応：** 原論文が扱う系は厳密には「1方向のみ有限な薄膜（$n$層×無限大）」であり、本課題の $L\times L$ 系とは設定が異なる。ただし周期境界条件のもとでスケーリング形式が指数関数的に速く収束することが示されており（同論文 p.1518）、$L\times L$ 系への適用を正当化している。この枠組みを $L\times L$ 系に適用したのはBinder以降の研究による。

### ビンダークロスによる $T_c$ 決定

ビンダーキュムラントは $\langle m^4\rangle$ と $\langle m^2\rangle^2$ の比なので $L$ 依存の係数が打ち消されて $p=0$ となり：

$$U_L(T) = \tilde{f}\!\left[(T - T_c) L^{1/\nu}\right]$$

この性質から、**異なるシステムサイズ $L$ での $U_L(T)$ 曲線は $T = T_c$ で交差する（ビンダークロス）**。

```
U_L
 2/3 |         __________
     |        /   L=64
     |       /  __________
     |      / /   L=32
     |   __/ /
     |  /   /  ← 交差点 = T_c
     | /   /
   0 |____________________________→ T
              T_c ≈ 2.269
```

### 2次元イジング模型の臨界指数

Onsagerの厳密解から以下が得られている：

| 指数 | 値 | 意味 |
|---|---|---|
| $\nu$ | $1$ | 相関長 $\xi \sim \|T-T_c\|^{-1}$ |
| $\beta$ | $1/8$ | 磁化 $m \sim (T_c-T)^{1/8}$ |
| $\gamma$ | $7/4$ | 磁化率 $\chi \sim \|T-T_c\|^{-7/4}$ |

$\nu=1$ なので、スケーリング変数は $(T-T_c)L$ とシンプルになる。

この交点を外挿することで $T_c$ を決定する。

## 5. 計算手順のまとめ

1. **実装とユニットテスト（各機能と並行して実施）**  
   各機能を実装する都度、小さな $L$（$L=2, 4$ 等）でテストする：
   1. エネルギー計算：全スピン整列配置で $H = -2JL^2$ となることを確認
   2. メトロポリス更新：高温で磁化→0、低温で磁化→$\pm 1$ に収束することを確認
   3. 物理量の計算：$\langle m^2\rangle$、$\langle m^4\rangle$ 等が正しく集計されることを確認
   4. $O(L^2)$ スケーリング：$L$ を2倍にするごとに1 MCSの計算時間が約4倍になることを確認

2. **本計算**  
   $L = 8, 16, 32, 64$ 等、複数のシステムサイズと $T_c \approx 2.269$ 周辺の温度グリッドを設定する。各 $(L, T)$ の組み合わせに対して以下を実行する：
   1. スピン配置を初期化（高温ではランダム、低温では全スピン整列など）
   2. $N_{\rm therm}$ MCS のメトロポリス更新で熱化（初期配置の影響を除く）。熱化中もスピン反転の試みは行うが物理量は記録しない。系はこの過程でボルツマン分布へ収束していく。
   3. $N_{\rm meas}$ MCS にわたって $m$、$m^2$、$|m|$、$m^4$ を各ステップで記録。「平衡後に1回だけ記録」では1配置の値しか得られず統計的に無意味。平衡後も系はボルツマン分布上をランダムウォークし続けるため、その軌跡を多数記録して時間平均を取ることで熱平均を近似する。
   4. 熱平均 $\langle m\rangle$、$\langle m^2\rangle$、$\langle|m|\rangle$、$\langle m^4\rangle$ を計算
   5. 磁化率 $\chi$ およびビンダーキュムラント $U_L$ を計算

3. 各 $L$ について $\langle|m|\rangle$、$\chi$、$U_L$ を温度 $T$ の関数としてプロットする。
4. ビンダーキュムラントの交差点から $T_c$ を推定し、Onsager解と比較する。
5. 計算時間の $L$ 依存性を測定し、$O(L^2)$ スケーリングを確認する。

---

## 6. 実装上の注意点

- **周期境界条件**：インデックス計算に `%` 演算（mod）を用いる。
- **初期配置**：高温ではランダム配置、低温では全スピン整列配置が収束を速める。
- **統計誤差**：自己相関時間を考慮し、jackknife 法や bootstrap 法で誤差を推定する。
- **乱数**：メルセンヌ・ツイスタ等の品質の高い擬似乱数生成器を用いる。
- **$O(L^2)$ 確認**：$L$ を2倍にするごとに計算時間が約4倍になることを確認する。
